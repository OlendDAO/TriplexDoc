# 合成资产

Triplex协议支持多种类型的合成资产，用户可以通过超额抵押机制铸造这些资产，并在去中心化交易平台上进行交易。

<Callout type="info">
  合成资产是锚定某些标的资产价值的代币，不需要实际持有底层资产，而是通过智能合约和价格预言机来追踪价格。
</Callout>

## 铸造合成资产流程

<Steps>
### 存入抵押品
将加密资产存入Triplex智能合约作为抵押品

### 选择要铸造的合成资产
从支持的合成资产列表中选择您想要铸造的资产类型

### 设定铸造数量
根据您的抵押品价值和当前抵押率要求，设定要铸造的合成资产数量

### 确认交易
确认交易详情并在钱包中签名提交
</Steps>

## 抵押要求

为了确保系统的安全性和稳定性，Triplex实施严格的超额抵押机制：

<Callout type="warning">
  所有合成资产都需要至少150%的抵押率。不同资产类型可能有不同的最低抵押率要求。
</Callout>

| 合成资产类型 | 最低抵押率 | 清算抵押率 |
| :--- | :--- | :--- |
| 加密货币 (sBTC, sETH) | 150% | 125% |
| 法币 (sUSD) | 200% | 175% |
| 原生代币 (sAPT) | 175% | 150% |
| 现实世界资产 | 250% | 225% |

## 风险管理

<Callout type="error">
  合成资产持有者需要密切关注抵押率变化，如果抵押率低于清算阈值，抵押品可能被清算。
</Callout>

为了帮助用户管理风险，Triplex提供以下功能：

- 实时抵押率监控
- 抵押率预警通知
- 一键追加抵押品
- 部分还款功能
