<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">

  </summary>
  <div className="mt-4">

  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    如何铸造合成资产？
  </summary>
  <div className="mt-4">
    铸造合成资产的基本步骤：
    
    1. 连接您的钱包到Triplex平台
    2. 存入抵押品（支持BTC、ETH、USDC等）
    3. 选择要铸造的合成资产类型
    4. 确定铸造数量（需符合最低抵押率要求）
    5. 确认交易并签名
    
    铸造后，合成资产会直接发送到您的钱包中，您可以持有、交易或用于其他DeFi活动。
  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    什么是抵押率，如何计算？
  </summary>
  <div className="mt-4">
    抵押率是指您存入的抵押品价值与铸造的合成资产价值之比，计算公式为：
    
    ```
    抵押率 = (抵押品价值 / 合成资产价值) × 100%
    ```
    
    例如，如果您存入价值1500美元的ETH，并铸造价值1000美元的sBTC，您的抵押率为150%。
    
    Triplex要求不同合成资产有不同的最低抵押率（通常在150%-250%之间），以确保系统安全性。当抵押率低于清算阈值时，您的头寸可能面临清算风险。
  </div>
</details>

## 交易与流动性

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    如何交易合成资产？
  </summary>
  <div className="mt-4">
    在Triplex平台上交易合成资产：
    
    1. 连接您的钱包
    2. 导航至"交易"页面
    3. 选择交易对（如sBTC/sUSD）
    4. 输入交易数量
    5. 确认交易详情
    6. 签名交易并等待确认
    
    Triplex的交易平台支持合成资产之间的交易，以及合成资产与原生资产之间的交换。
  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    如何提供流动性？
  </summary>
  <div className="mt-4">
    提供流动性的步骤：
    
    1. 连接钱包
    2. 访问"流动性"页面
    3. 选择要提供流动性的资产对
    4. 输入两种资产的数量
    5. 确认并签名交易
    
    提供流动性后，您会收到LP代币，代表您在流动性池中的份额。LP代币可以持有以赚取交易费分成，也可用于质押获得额外的TRT奖励。
  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    什么是无常损失，如何规避？
  </summary>
  <div className="mt-4">
    无常损失是指当您提供流动性时，由于资产价格相对变化导致的潜在损失，与单纯持有这些资产相比。
    
    规避无常损失的策略：
    
    - 为价格相关性高的资产对提供流动性（如稳定币对）
    - 在波动性较低的市场环境中提供流动性
    - 长期提供流动性，让交易费收入抵消无常损失
    - 使用Triplex的单边流动性池（某些特定池不受无常损失影响）
    - 考虑使用Triplex的无常损失保险机制（即将推出）
    
    您可以使用我们的[无常损失计算器](/provideLiquidity/calculator)评估潜在风险。
  </div>
</details>

## 安全与风险

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    Triplex如何确保协议安全？
  </summary>
  <div className="mt-4">
    Triplex通过多层安全措施保护协议和用户资产：
    
    - **代码审计**：多家知名安全公司进行全面审计
    - **形式验证**：对核心合约进行数学形式验证
    - **漏洞赏金计划**：鼓励白帽黑客发现漏洞
    - **多重签名**：关键操作需要多方批准
    - **时间锁**：关键参数变更有延迟实施期
    - **渐进式去中心化**：逐步将控制权移交给社区
    - **限额系统**：限制单一操作的规模
    - **异常监控**：实时监控系统异常行为
  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    什么是清算，如何避免？
  </summary>
  <div className="mt-4">
    清算是当您的抵押率低于清算阈值时触发的过程，会导致您的部分抵押品被卖出以偿还债务。
    
    避免清算的方法：
    
    - 保持足够高的抵押率（建议至少比最低要求高20%）
    - 设置抵押率警报通知
    - 在市场波动加剧时主动增加抵押品或偿还部分债务
    - 使用稳定币作为抵押品以减少价格波动风险
    - 分散使用不同类型的抵押品
    - 考虑使用Triplex的清算保护工具（即将推出）
  </div>
</details>

<details className="rounded-lg border border-gray-200 p-4 dark:border-neutral-800 my-4">
  <summary className="font-medium cursor-pointer">
    预言机如何工作？如何确保价格准确性？
  </summary>
  <div className="mt-4">
    Triplex使用多来源预言机网络确保价格数据的准确性和安全性：
    
    - **多来源聚合**：从多个预言机服务获取数据
    - **中位数过滤**：过滤异常值，使用中位数价格
    - **时间加权平均**：降低价格操纵风险
    - **心跳检查**：确保价格数据的时效性
    - **偏差阈值**：监测价格大幅波动的异常情况
    - **失败备援机制**：预言机失效时的备用方案
    
    此外，Triplex与多个知名预言机网络合作，确保价格数据的可靠性和防篡改性。
  </div>
</details>
