# Triplex

Triplex是在Aptos和其他兼容链上的去中心化流动性提供协议，提供模块化架构和灵活的流动性提供系统。它引入了一种创新的DeFi框架，解决了传统链上衍生品平台面临的流动性获取和基础设施扩展等挑战。

通过Triplex，用户可以体验更高效的资本利用率、更灵活的风险敞口以及全新的互操作性体验。

## 它是如何工作的？

```jsx
import { useAccount } from '@triplexjs/hooks'

// 获取账户余额，流动性、仓位等
const { data } = useAccount()
```

Triplex的前端集成实现了直观的用户体验。通过使用我们的React hooks，您可以轻松地将Triplex功能集成到您的应用程序中。

```jsx
import { useMarket } from '@triplexjs/hooks'

// 获取市场数据
const { data } = useMarket('BTC-trtUSD')
```

## 生态系统

Triplex是由多个核心组件构成的：保险库、资金池、市场和奖励分配器。这些组件共同工作，为用户提供了高度灵活的DeFi体验。

我们不断发展的生态系统包括：

- 永续合约交易
- 杠杆代币
- 合成资产
- 治理
- 各种集成选项
