# Triplex V3 概述

## Perps V3 有哪些新特性？

Triplex V3 在先前版本的基础上进行了多项重要升级，为交易者提供更强大、更灵活的交易体验：

- **原生跨保证金**：这一特性提供了交易者使用其整个投资组合作为抵押品的能力，增强了交易效率和风险管理。

- **更好的定价**：在Perps V3中，订单提交时的价格—即您提交订单的那一刻的价格—就是您交易执行的价格。这一升级消除了因执行延迟导致的价格滑点的挫折感。

- **失败订单替换**：执行失败的订单不再需要"取消"，一旦失败，交易者可以简单地下另一个订单，它会自动替换失败的交易。

- **重新设计的用户体验**：Perps V3引入了对导航、交易流程和数据呈现的显著改进。这一全面改造专注于直观性和效率，确保新手和经验丰富的交易者都能充分利用平台的能力。

## 引擎盖下是什么？

### NFT账户

NFT账户是V3中账户管理的创新方法，提供了更灵活、更用户友好的方式来管理交易活动。作为不可替代代币(NFT)，您的交易账户可以：

- 在不同钱包之间转移
- 授权给其他地址进行管理
- 与您的其他Web3资产一起可视化

<Callout type="warning">
账户作为NFT意味着您可以转移它，但也意味着您需要小心保管。如果您丢失了对拥有账户NFT的钱包的访问权限，您将无法再访问该账户。
</Callout>

### 超级市场

超级市场是核心风险管理工具，允许在不限制市场产品广度的情况下提供原生跨保证金。每个"超级市场"包含一个或多个共享抵押品的永续市场。

超级市场的设计允许：
- 高效的跨保证金计算
- 隔离的风险环境
- 灵活的市场分组

### 智能保证金 V3

智能保证金V3(SMv3)延续了其前辈的传统，旨在通过提供先进的工具来提升交易体验，用于交易Triplex衍生品。

智能保证金V3提供：
- 改进的保证金计算
- 更高的资本效率
- 增强的风险管理功能
- 更灵活的抵押品选项

## 接下来会有什么？

Triplex Exchange在Base上才刚刚开始。在未来几个月，用户可以期待更多的功能和改进。

### 混合订单簿系统

备受期待的订单簿将与现有的AMM模型并行运行，为交易者提供对订单的更多控制，更好的价格发现和减少滑点。

Chainlink数据流：来自Chainlink的低延迟预言机将提高价格馈送的速度和准确性，增强Triplex市场的性能。

<Callout type="info">
混合系统将结合AMM的流动性优势和订单簿的精确性，为交易者提供前所未有的灵活性。
</Callout>

### 额外的保证金资产

随着Triplex扩展其多抵押品系统，更多的资产如稳定币、治理代币和其他ERC-20将被添加为保证金选项。

请关注这些更新，因为Triplex继续发展以赋能交易者。
